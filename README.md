## 项目介绍  

一如既往的兴趣使然，学习一下`flutter`，本项目更多的是借鉴了前辈们的项目中的组件，当然也有自己发挥的部分，比如登录、注册、聊天页面~哈哈哈~~~大家可以用作参考和学习哦～～
